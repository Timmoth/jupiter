Visualizing Jupiter and its four largest moons, Io, Europa, Ganymede and Callisto

Clone

```
git clone https://github.com/Timmoth/jupiter.git
```

Build

```
npm run build
```

Run

```
npm install http-server -g
http-server
```

App will be available at http://localhost:8080/
